# VPet 知识库（可扩展）

## 项目现状
当前 Flutter 版本是桌宠重构版，已实现：
- 透明无边框 macOS 桌宠窗口
- 单进程双窗口桌宠（两只同进程，便于互动）
- 基础动作：default / move / sleep
- 触摸区交互：head / body / pinch

## 动作节奏
- default：主帧停留 2-3 秒，闭眼帧短闪
- sleep：帧切换明显放慢，接近呼吸节奏
- move：仅在 move 动作时窗口移动

## 常用操作
- 左键点击宠物会弹出菜单
- 菜单支持：AI 对话 / 安排日程 / 和另一只互动 / 显示或隐藏触摸区 / 退出

## 角色性格
Mu-1:
- 更主动、偏外向，喜欢先发起话题
- 回复短句，语气轻快
- 倾向先行动后表达

Mu-2:
- 更稳重、偏理性，喜欢确认计划
- 回复较完整，语气温和
- 倾向先评估后行动

## 日程规则
- 日程包含时间段、话题、动作列表
- 在日程生效时间内会随机触发动作与双宠对话
- 动作字段支持预留名（如 `reserved_chat`），后续可映射新动画

## AI 对话说明
- 若设置 `OPENAI_API_KEY`，对话会调用在线模型
- 若未设置，会回退到本地知识库检索回答
- 可通过环境变量 `OPENAI_BASE_URL` 和 `OPENAI_MODEL` 覆盖默认值

## 计划中的能力
- 行为图谱与更多状态迁移
- 事件记忆和长期对话上下文
- 更丰富动作资源与语音能力
